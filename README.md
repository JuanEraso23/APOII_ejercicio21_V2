# APOII

Curso de Algoritmos y Programación II.

## Descripción

Programa en Java que implementa un menú interactivo con diferentes operaciones matemáticas utilizando estructuras repetitivas y condicionales.

El sistema permite:
- Realizar una suma continua de números enteros.
- Calcular el promedio de 10 números.
- Salir del programa mediante una opción del menú.

El ejercicio fue desarrollado utilizando diferentes enfoques:
- Menú con `switch`.
- Menú con estructuras `if` y `else if`.

Características implementadas:
- Ciclos repetitivos (`while` y `for`).
- Estructuras de selección.
- Operaciones matemáticas.
- Validación básica de opciones.

Este ejercicio hace parte de los fundamentos de programación estructurada y desarrollo de menús interactivos en Java.
